# Agentic-AutoML

Agentic AutoML is a design-first repository for building machine learning systems with AI agents. The project is centered on a structured loop for dataset understanding, experiment planning, code generation, execution, review, routing, and memory.

## Current Direction

The current plan is to start with a narrow, reliable vertical slice:

- predefined datasets
- tabular supervised ML
- deterministic data profiling
- planner -> code -> execution -> review loop
- persistent project memory
- knowledge-base support for reusable ML tactics

This repo currently captures the product definition and implementation scaffolding before code is added.
